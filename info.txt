1. dodać na końcu pliku C:\xampp\apache\conf\extra\httpd-vhosts.conf

<VirtualHost *:80>
	ServerName projekt.lovalhost
	DocumentRoot "C:\xampp\htdocs\projekt\public"
	<Directory "C:\xampp\htdocs\projekt\public">
        AllowOverride All
        Require all granted
    </Directory>
</VirtualHost>




2. zrobić 
composer install
npm install
npm run build

jeśli nie można zrobić composer install to trzeba usunąć plik composer.lock a następnie wpisać wpisać composer update
